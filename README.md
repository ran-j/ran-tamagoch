# Tamagotchi Game

This is a simple CLI-based Tamagotchi game implemented in Go.

## Getting Started

### Prerequisites

- [Go](https://golang.org/dl/) (version 1.16 or later)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/ran-j/ran-tamagotchi
```

2. Build the project:

```bash
make build
```

3. Run the game:

```bash
./tamagotchi
```
